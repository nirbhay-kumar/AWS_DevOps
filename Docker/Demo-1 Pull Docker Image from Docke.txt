***********Demo-1: Pull Docker Image from Docker Hub and Run it

Step-1: Verify Docker version and also login to Docker Hub
docker --version
docker login

Step-2: Pull Image from Docker Hub

docker pull rajendrawagh/sampleimage:v1-release

Check that with following command

docker images

Step-3: Run the downloaded Docker Image & Access the Application

docker run --name app1 -p 80:80 -d rajendrawagh/sampleimage:v1-release

check this on browser http://localhost/

Step-4: List Running Containers
docker ps   		list of running containers
docker ps -a		list of all containers ( running + stopped )
docker ps -a -q		Ids of all containers ( running + stopped )
docker ps -q		Ids of all running containers


Step-5: Connect to Container Terminal
docker exec -it <container-name> /bin/sh
ls
exit

Step-6: Container Stop, Start
docker stop <container-name>
docker start  <container-name>

Step-7: Remove Container
docker stop <container-name> 
docker rm <container-name>

Step-8: Remove Image
docker images
docker rmi  <image-id>

Step 9: Remove all Images
docker rmi $(docker images -a -q)
