Run following commands in project folder where you have Dockerfile, requirements.txt and source code

docker build -t my-docker-app .

docker run --name app4 -p 80:5000 -d my-docker-app

After the build completes, tag your image so you can push the image to this repository:
docker tag my-docker-app 600330284614.dkr.ecr.us-east-1.amazonaws.com/demo:latest


Run the following command to push this image to your newly created AWS repository:
docker push 600330284614.dkr.ecr.us-east-1.amazonaws.com/demo:latest
